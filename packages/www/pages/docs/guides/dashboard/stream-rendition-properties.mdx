# How to set stream rendition properties in the Livepeer.com dashboard

The width, height and frame rate are not currently configurable in the user
interface.

Livepeer.com provides a default adaptive bitrate (ABR) ladder. The Livepeer.com
default ABR ladder provides seamless playback for viewers with various
combinations of available bandwidth, device performance, and network conditions.
