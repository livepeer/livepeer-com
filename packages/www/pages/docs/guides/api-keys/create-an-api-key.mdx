# How to create a Livepeer.com API key

If you havenâ€™t already,
[create an account on Livepeer.com](/docs/guides/account/create-an-account).

1. Open your Livepeer.com account and navigate to the **API key list page**,
   [https://livepeer.com/app/user/keys](/app/user/keys)
1. Click the button to create a new key.
1. Input a name for your key and **create** the key.
1. When prompted, **copy** and **secure** the API key. This key is a secret, and
   for your security Livepeer.com will not show it to you again.
1. Once you have secured your API key, close the prompt.
