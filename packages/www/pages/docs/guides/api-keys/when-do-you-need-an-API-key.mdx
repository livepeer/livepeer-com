# When do you need an Livepeer.com API key?

You need an API key to live stream with the Livepeer.com API..

You do not need to create an API key for the Livepeer.com dashboard-only workflow.

Next, learn [how to create a Livepeer.com API key](/docs/guides/api-keys/create-an-api-key).
