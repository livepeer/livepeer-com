# How to setup a CNAME record for the Livepeer.com global ingest URL

The Livepeer.com global ingest URL is `rtmp://rtmp.livepeer.com/live/<stream-key>`.

You can CNAME `rtmp.livepeer.com` to customize that URL to something with the your brand, for example `rtmp.companyname.io`. You can configure this with your DNS provider.
